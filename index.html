Bespoke Post

e-commerce

Bespoke Post is an online retailer with a hybrid store/subscription model. Standard e-commerce software was not suitable to the specifics of their business model, so we created a custom order management system and web site using Ruby on Rails. The application integrated many external APIs, including payment processors, mailing lists, and warehouses. Since performance was a concern, we built multiple levels of caching and dynamic content loading, and hosted the application on scalable cloud infrastructure. It included a responsive design optimized for mobile, tablet, and desktop, with retina-optimized photography and marketing insights via analytics.

Nick built a platform for Bespoke Post which has reliably processed millions of dollars in transactions. We worked closely to execute my creative vision and I was extremely satisfied with the results.
–Peter Posen, Creative Director, Bespoke Post
  • Date: 2012-2014
  • Company: Bespoke Post
  • Category: E-Commerce
